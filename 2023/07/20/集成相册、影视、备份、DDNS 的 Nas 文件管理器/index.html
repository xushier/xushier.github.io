集成相册、影视、备份、DDNS 的 Nas 文件管理器

发表于 2023-07-20 更新于 2023-11-24 分类于 Unraid ， Docker Waline： 阅读次数： 本文字数： 1.1k 阅读时长 ≈ 4 分钟
包含照片备份、智能相册、影视刮削播放、文件备份以及 DDNS 的 Nas 文件管理器。

视频

功能

Nascab 是一个文件管理器，在基本的文件浏览管理、上传下载的基础上，集成了照片备份、智能相册、影视文件展示及刮削播放、文件备份以及 DDNS 等功能。

安装

镜像：ypptec/nascab

docker run

如果想要使用 docker-compse 安装，参考如下配置。

也可使用 https://www.composerize.com 快速转换。

unraid

使用

安装完成后访问 6592 端口，首先进行管理员账号创建。

管理员账号的创建是任意的，并没有绑定邮箱或手机的操作，后续如果密码忘记不能通过邮箱或手机号找回，只能重置。两步验证采用的是密保问题和答案，且是必填项。

管理员账号创建完成之后进入 Nascab 的主页面。主要包含以下几个功能。

文件浏览

文件浏览支持基本的上传下载和共享，共享支持设置密码和有效期。

相册管理

在相册管理界面，添加一个照片文件的来源之后，程序就会开始扫描并且进行照片中物体和人物（前提是先在 AI 相册选项中打开）以及位置的分析。

时间轴选项按照时间线对照片进行展示。

我的足迹选项根据照片内置的位置信息将照片显示在地图上。高缩放等级地图需付费解锁。

智能相册选项可以根据智能日期（节假日、纪念日及日期区间等）、地理围栏、设备型号以及照片文件的其他元数据诸如大小、宽高、路径、文件名等进行筛选，将满足结果的照片归入指定命名的相册。

普通相册选项类似文件夹功能。

AI 相册选项包含人物识别、物体识别和重复照片扫描。扫描效果不是很好。

最近、回收站和文件来源，顾名思义，不再赘述。

影视管理

影视库界面排版粗糙，海报刮削需要付费，且刮削效果不好。

智能影集类似智能相册。

影视播放方面，浏览器支持的 H264 文件是直推播放，265 编码的文件不支持硬件解码，使用 CPU 解码，占用很高。

文件夹共享

文件夹共享功能采用的是 WEBDAV 服务和 FTP 服务。

加密空间

加密空间功能可以对指定的文件夹设置一个使用 AES256 算法加密的密码，使用加密空间中的文件时会实时加解密。

安全设置

安全设置里可以设置 IP 黑白名单，以及登陆失败的 IP 封禁策略、查看日志等。

文件备份

文件备份只支持移动端，电脑端的备份只是将电脑本机的文件进行移动。

DDNS

DDNS 和开机启动这两个功能感觉上是为 Windows 客户端设计的，docker 端没有什么用。

移动端

移动端看起来像是 Web 端的套壳，所有功能和布局基本一样，多了一个相册备份，以及文件备份可以备份到服务端。

总结

  • 文件浏览还需要细节上的改进。
  • 相册识别效果不好。
  • 影视管理不能硬解播放，且收费的刮削功能体验不好。
  • 文件备份功能只能用于手机端。
  • 手机端软件较为粗糙，相册备份功能还可以。拥有 Windows 服务端，对使用 Windows 作为 Nas 的小白较为友好。

软件需要实现的功能很好，但目前较为粗糙的情况下部分功能还需收费。希望作者能尽快改进，达到可用的状态。
